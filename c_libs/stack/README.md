# C stack implementation

(Test example in `stack/src/main.c`)

## How to build (cmake version 3.2+)

> cd stack  
> mkdir build  
> cd build  
> cmake ..

You will have shared libraries `libStack.so` at `stack/build/src/lib`.

Header files to include are in `stack/src/include`

## Run test example

> cd stack/build  
> ./app
