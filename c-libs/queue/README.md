# C queue implementation

(Test example in `queue/src/main.c`)

## How to build (cmake version 3.2+)

> cd queue  
> mkdir build  
> cd build  
> cmake ..

You will have shared libraries `libQueue.so` at `queue/build/src/lib`.

Header files to include are in `queue/src/include`

## Run test example

> cd queue/build  
> ./app
